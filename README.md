# KARA:Arts 4차 단기선교 · 기도카드 웹페이지 🙏

노션 선교 안내 페이지를 바탕으로 만든 **기도카드 웹페이지**입니다.
- 📖 선교 소개 · 팀원 · 기도제목을 한 페이지에 담았어요
- 💌 방문자가 **응원·기도 댓글**을 남길 수 있어요
- 📲 페이지 안에 **QR코드**가 들어 있어 인쇄·공유가 쉬워요

```
prayer-card/
├── index.html              ← 웹페이지 본체 (이거 하나면 동작)
├── apps-script-backend.gs  ← 응원댓글 저장용 백엔드 코드
└── README.md               ← 이 파일
```

---

## 1단계 · 지금 바로 미리보기

`index.html` 파일을 더블클릭해 브라우저로 열면 바로 볼 수 있어요.
이 상태(미리보기 모드)에서는 댓글이 **내 기기에서만** 보입니다.

## 2단계 · 인터넷에 올리기 (무료, 가입 간단)

QR로 다른 사람에게 전달하려면 인터넷 주소가 필요해요. 셋 중 편한 걸 고르세요.

| 방법 | 난이도 | 방법 |
|------|--------|------|
| **Netlify Drop** | ⭐ 가장 쉬움 | https://app.netlify.com/drop 에 `prayer-card` 폴더를 드래그 → 끝 |
| **GitHub Pages** | ⭐⭐ | 깃허브 저장소에 올리고 Settings → Pages 활성화 |
| **Vercel** | ⭐⭐ | vercel.com 에서 폴더 업로드 |

올리고 나면 `https://...` 주소가 나와요. **그 주소로 접속하면 QR코드도 자동으로 그 주소를 가리킵니다.**

## 3단계 · 응원댓글을 모두가 함께 보이게 (선택, 5분)

여러 사람의 댓글을 함께 모으려면 저장 공간이 필요해요. 구글 계정만 있으면 무료로 됩니다.

1. [sheets.google.com](https://sheets.google.com) 에서 빈 스프레드시트 생성
2. 메뉴 **확장 프로그램 → Apps Script**
3. `apps-script-backend.gs` 내용을 전부 붙여넣고 저장
4. **배포 → 새 배포 → 웹 앱**
   - 실행 계정: **나**
   - 액세스 권한: **모든 사용자** ★중요
5. 나온 **웹 앱 URL**(`.../exec`)을 복사
6. `index.html` 을 열어 윗부분의
   ```js
   const BACKEND_URL = "";
   ```
   따옴표 안에 그 URL을 붙여넣고 저장 → 다시 업로드

이제 누가 어디서 접속해도 댓글이 함께 쌓이고, 내용은 내 구글 시트에 차곡차곡 저장돼요.

## QR코드는?

페이지 맨 아래 **‘기도카드 나누기’** 섹션에 QR코드가 자동으로 표시됩니다.
- 인쇄: 그 화면에서 **🖨️ 카드 인쇄** 버튼 → 응원/입력란은 빠지고 카드용으로 깔끔하게 인쇄돼요
- 따로 이미지가 필요하면 알려주세요. 최종 주소로 된 QR PNG 파일을 만들어 드릴게요.
